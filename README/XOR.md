# XOR  (Exclusive OR)

## Self-inverse property

```
x XOR x = 0
x XOR 0 = x
```

## Commutativity

```
x XOR y = y XOR x
```

## Associativity
`(a XOR b) XOR c` equals to `a XOR (b XOR c)`

## Distributive property of XOR over addition

```
A XOR (B + C) = (A XOR B) + (A XOR C)
```

## XOR Equation
Equation like `a = b ^ x` or `a = x ^ b` can be solved like: `x = a ^ b`
